Quick Start

Three lines. That's it.

Install the SDK, create a client, send a message. No configuration, no setup wizards.

TypeScript send-sms.ts
import { PingText } from "pingtext";

const pt = new PingText(process.env.PINGTEXT_API_KEY);

await pt.messages.send({
  to: "+15551234567",
  body: "Your order has shipped! 📦",
});

// Check delivery status
const status = await pt.messages.get(msg.id);
console.log(status.status); // "delivered"
Python send_sms.py
import pingtext
import os

pt = pingtext.PingText(os.environ["PINGTEXT_API_KEY"])

msg = pt.messages.send(
    to="+15551234567",
    body="Your order has shipped! 📦",
)

# Check delivery status
status = pt.messages.get(msg.id)
print(status.status)  # "delivered"
